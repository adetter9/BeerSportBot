# SportPivBot
Telegram-бот для учёта алкогольного баланса с пересчётом по индивидуальным формулам спорта. Поддерживает несколько типов напитков, персональные коэффициенты и хранение данных в SQLite.

Есть в тг, имя пользователя: @SportPivBot

Бот считает доступный баланс в литрах выбранного напитка. Баланс может быть отрицательным, но это значит, что его все равно нужно будет отработать.

• «Выпить» уменьшает баланс.

• «Спорт» увеличивает баланс по текущей формуле.

• «Сменить напиток» переключает напиток и пересчитывает баланс.



Команды:

/start — меню

/info — справка

/change — сменить напиток

/setrate X — задать формулу для текущего напитка (литров за 60 минут)
